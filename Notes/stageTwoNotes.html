CLI Command Line Cheetsheet

  • ls: List directory
  • cd: Change directory
  • pwd: Print the location path
  • open: Opnes a file on
  • touch: Creates a new file if one dosen't already exist or udates the file's time stamp
  • mkdir: Makes a new directory(folder)
  • rm: Removes a file
  • rmdir: Removes an empty directory
  • rm -rf: Removes a directory and its contenents (Without confirmation)
